# Contributing to Pivot Bridge

Thank you for your interest in contributing to the Pivot Bridge! As the Pivot Bridge is a GPL-licensed project that interfaces with a proprietary backend, all code contributions must comply with the following simple guidelines:

1.  **License:** By submitting a pull request, you agree that your contributions will be licensed under the **GNU General Public License, Version 3 (GPLv3) or any later version.**
2.  **Author Rights:** You certify that you are the original author of the code and have the right to contribute it under the GPL.
3.  **Code of Conduct:** Please adhere to respectful and professional community standards.
